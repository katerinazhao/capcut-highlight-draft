# CapCut Highlight Draft

## 中文

`CapCut Highlight Draft` 是一个用于把会议录屏、访谈录屏或长视频素材整理成高光片段，并生成可直接用于剪映/CapCut 的工程草稿的 Skill。

### 这个项目会做什么

- 从录屏或会议素材中筛选高价值片段
- 生成可执行的剪辑计划
- 输出切好的视频片段
- 生成剪映/CapCut 可用的 draft 工程目录

### 目录内容

- `SKILL.md`：Skill 的核心定义和工作流
- `agents/openai.yaml`：显示名和默认调用配置




## English

`CapCut Highlight Draft` is a Skill for turning meeting recordings, interview recordings, or long-form video into highlight clips and a draft project that can be used in CapCut/Jianying.

### What it does

- Selects high-value segments from long recordings
- Creates an actionable edit plan
- Produces cut video clips
- Builds a CapCut/Jianying draft project directory

### Contents

- `SKILL.md`: core skill definition and workflow
- `agents/openai.yaml`: display name and default invocation config
